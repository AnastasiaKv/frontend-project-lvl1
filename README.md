## Setup

```sh
make install
```

## Run

```sh
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```

#### or

## Install module locally and run

```sh
npm link

brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
